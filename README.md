Github를 이용하는 이유
=====

- 제가 어디서나 github에 커밋가능한 환경하에 있고
- 펀치카드 같은 그래프를 보여줘서.. 얼마나 빼먹었나 보이는것도 좋고
- 혹시 함께 하는 분이 있다면 경쟁을 할 수 있어서 좋기 때문에..
- 서로 운동하다 안하는거 같으면 상호 이슈로 등록하여 assign 할 수 있습니다. 커밋해서 리졸부해야합니다.
